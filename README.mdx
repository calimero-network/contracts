# Calimero Network Smart Contracts

[![Discord](https://dcbadge.vercel.app/api/server/wZRC73DVpU?style=flat&theme=default-inverted)](https://discord.gg/wZRC73DVpU)
[![License](https://img.shields.io/badge/License-Apache--2.0-blue)](#license)
[![Issues - calimero](https://img.shields.io/github/issues/calimero-network/contracts)](https://github.com/calimero-network/contracts/issues)

This repository contains the smart contracts that power the [Calimero Network](https://github.com/calimero-network/core), a multi-chain infrastructure solution.

## Project Structure

The repository is organized by blockchain platform, with each platform having its own set of contracts and documentation:

- [Ethereum Contracts](contracts/ethereum/README.md)
- [NEAR Protocol Contracts](contracts/near/README.md)
- [Internet Computer Protocol Contracts](contracts/icp/README.md)
- [ZkSync Era Contracts](contracts/zksync/README.md)

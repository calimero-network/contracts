Supporting contracts for the [Calimero Network](https://github.com/calimero-network/core).

## Local setup

Dependencies:
- [`rustc` and `cargo`](https://www.rust-lang.org/tools/install) - required for rust contracts
- [PocketIC](https://github.com/dfinity/pocketic) - required for ICP contracts

To build and test rust contracts locally, run the following commands:

```bash
./scripts/build-rust.sh
cargo test
```
